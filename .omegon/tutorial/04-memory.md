Watch the **memory strings** in the left side of the instrument panel — the lines below the context bar, connected by a tree connector (│├└).

I'm going to store some facts about this project. Each store operation "plucks" the memory string — you'll see waves travel **rightward (→)**, representing data flowing into memory.

Store these 3 facts:
1. "This project is a Rust tutorial sandbox for learning Omegon features"
2. "The analysis module counts words, lines, and characters in text"
3. "The config module provides a Config struct with name and verbose fields"

After storing, tell the operator: "Each store sent a rightward wave (→) — data flowing into memory. The string vibrates and dampens naturally."

Now recall the project architecture. After: "That recall sent waves **leftward (←)** — data flowing out of memory. Store goes right, recall goes left. You'll develop intuition for this over time."
