# Omegon Interactive Demo — Phase 1

Welcome the operator to Omegon. Explain what they're looking at:

The instrument panel at the bottom of the screen has two sections:
- **Left — Inference**: a context bar (gradient fill showing how much context is used) with glitch characters that appear during thinking. Below it, tree-connected memory "strings" — one per linked mind, showing fact counts.
- **Right — Tools**: a sorted list of tools ordered by most recently used. Each shows a recency bar and time since last call.

The engine panel on the bottom-left shows the model name, tier, thinking level, and context percentage.

After explaining, tell the operator:

"That's the cockpit. Let's make it light up! When you're ready, type **next** and I'll start reading files so you can watch the instruments respond."

**IMPORTANT: Do NOT do anything else. Do NOT read files. Do NOT proceed to any other phase. Just explain and wait.**

When the operator types "next", "continue", or "go", read the AGENTS.md file for the full phase list and execute the next phase only.
